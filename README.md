# Debt burndown calculator built in React inside Tauri

## Tech used

- Tauri
- Tauri Plugin-Store
- Vite with HMR (buzzy enough?)
- React + Typescript
- React Router
- Chart.js
- Finance.js

## Installation

- Meet all [Tauri requirements](https://tauri.app/v1/guides/getting-started/prerequisites)
- Install PNPM
- run `npm install` in root directory
